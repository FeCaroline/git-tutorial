Lista de comandos para Git

Comandos para iniciar

  • git config --global user.name "Fulana"
    Nome do usuário
  • git config --global user.email "Fulana@etc.com"
    E-mail do usuário

Comandos para navegar nos diretórios

  • ls/ dir
    listar diretórios
  • pwd
    present working directory (caminho atual)
  • cd
    change directory (ir para pasta diferente)
  • cd ~
    volta para pasta raiz
  • cd .
    volta uma pasta acima
  • cd nome-da pasta
    entrar em uma pasta
  • touch nome-arquivo.extensão
    criar novo arquivo

Comandos para adicionar, comitar, etc

  • git init
    avisa o git para ficar de olho na pasta
  • git status
    mostra o que foi modificado, adicionado e etc
  • git add nome-arquivo.html
    colocar na caixa a versão do arquivo
  • git add .
    colocar na caixa todos os arquivos
  • git commit -m "descrição versão"
    etiquetar com o que foi mudado na versão
  • git log
    timeline com todos os commits feitos

Comandos mais utilizados

  • git checkout --arquivo.txt
    voltar para a versão do último commit
  • git branch nome-branch
    criar um novo branch (linha do tempo separada da master)
  • git checkout nome-branch
    trocar de branch (antes foi só criado
  • git merge nome-branch
    juntar o branch atual com o principal (master)
